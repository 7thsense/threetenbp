README for JSR-310 OpenJDK challenge
2008-08-04

Please see the README for a general introduction to the sourcebase.

This file describes the version of the JSR-310 reference implementation
submitted as the final project for the OpenJDK challenge.

The original proposal:
http://mail.openjdk.java.net/pipermail/challenge-discuss/2008-March/000062.html

The subversion repository of the final project:
https://jsr-310.dev.java.net/svn/jsr-310/tags/OpenJDKChallenge

The javadoc of the final project:
https://jsr-310.dev.java.net/nonav/doc-2008-08-04/index.html

The JSR-310 home page:
https://jsr-310.dev.java.net/

The JSR-310 mailing list:
https://jsr-310.dev.java.net/servlets/SummarizeList?listName=dev

The JSR-310 wiki:
http://wiki.java.net/bin/view/Projects/DateTimeAPI
